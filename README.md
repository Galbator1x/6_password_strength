6_password_strength
===================

This script takes the input password and evaluates the complexity scale from 1 to 10.

Usage
-----

```
~$ python3 password_strength.py 
Enter the password: 123456qwe
Password strength: 1
Enter the password: sdf23!.,DSF
Password strength: 8
```
Press enter to exit.

Requirements
------------

- Python >= 3.4
